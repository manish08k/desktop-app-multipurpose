# Studio Suite

Studio Suite is a multi-purpose desktop productivity application built with Python that combines several essential tools into a single workspace.

## Features

### File Manager

* Browse files and folders
* Navigate directories
* Create folders
* Open and manage files

### Text Editor

* Create and edit text files
* Save and organize documents

### Media Player

* Play audio and media files

### Image Viewer

* View and manage images

### CSV Viewer

* Open and inspect CSV datasets

### Notes

* Create and manage quick notes

### Calculator

* Perform daily calculations

### Pomodoro Timer

* Productivity timer for focused work sessions

### Calendar

* Track dates and schedules

### Password Manager

* Store and manage credentials

### Color Picker

* Select and copy color values

### Terminal

* Execute commands directly from the application

## Tech Stack

* Python
* Tkinter
* PyInstaller

## Installation

```bash
git clone https://github.com/manish08k/desktop-app-multipurpose.git
cd desktop-app-multipurpose
pip install -r requirements.txt
python suite.py
```

## Build

```bash
pyinstaller StudioSuite.spec
```
